# twi-scanner

Для установки libft4222:
1) Перейти https://ftdichip.com/products/ft4222h/ во вкладку Downloads;
2) Скачать LibFT4222 Linux Library;
3) Распаковать архив;
4) Выполнить `sudo ./install4222.sh`;
5) **libft4222.h** скопируется в /usr/local/include, а
   **libft4222.so** скопируется в /usr/local/include;

Для deploy проекта запустить скрипт deploy/deploy_qt.sh
